# radroots-net-core

Network orchestration primitives and runtime interfaces for the Radroots SDK.

## Goals

- define stable interfaces for network configuration, assembly, and control
- keep network orchestration behavior deterministic across runtime environments
- support feature-gated Nostr client, key, and persistence integration surfaces
- provide reusable network primitives for higher-level Radroots crates
