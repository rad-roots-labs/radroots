# radroots-core

Core value types and unit semantics for the Radroots SDK.

## Goals

- define invariant-safe domain primitives for shared SDK use
- keep arithmetic and unit conversion deterministic across targets
- support no_std and std builds with feature-gated serialization and type export
- provide stable primitives for higher-level Radroots crates
