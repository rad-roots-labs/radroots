# radroots-types

Shared API result and error wrapper types for the Radroots SDK.

## Goals

- define stable success and error envelope interfaces across SDK surfaces
- keep envelope serialization behavior deterministic across API boundaries
- support feature-gated type export for external SDK consumers
- provide reusable wrapper primitives for higher-level Radroots crates
