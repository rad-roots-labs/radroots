# radroots_nostr_ndb

Nostrdb adapter primitives and runtime interfaces for the Radroots SDK.

## Goals

- define stable nostrdb config, ingest, query, and subscription interfaces
- keep nostrdb storage and query behavior deterministic across runtimes
- support runtime adapter surfaces for shared event store integration
- provide reusable nostrdb primitives for higher-level Radroots crates
